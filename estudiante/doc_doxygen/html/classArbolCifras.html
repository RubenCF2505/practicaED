Genera de manera recursiva las posibles soluciones del árbol.

A partir del nodo indicado, combina los números mediante operaciones básicas y genera nuevos nodos hijos hasta encontrar el objetivo o agotar combinaciones.

Parameters
actualNodo desde el cual generar los hijos.
objetivoValor que se desea obtener.
Returns
Cadena que representa el camino de la solución, o cadena vacía si no se encuentra solución desde ese nodo.
